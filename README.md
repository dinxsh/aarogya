## Setup ğŸ’»

1. **Install Expo CLI:**
   ```bash
   npm install -g expo-cli
   ```

2. Clone the repository:

   ```git clone https://github.com/dinxsh/aarogya.git```


3. Install dependencies:

   ```cd aarogya
   npm install
   ```

4. Start the project:

   ```expo start```
